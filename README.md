# PHP DEVELOPMENT ENVIRONMENT

I wil use this repository to save my php sandbox where I will use last php versions.

## Installation

First of all you have to clone the repository to your computer and open it with
VSCode.

In your VSCode open a new terminal (Cntrl+Shift+ñ) and write next command:

```bash
docker-compose up --build -d
```

Have Fun!!


## Docker Services
 1. Nginx:1.23.1
 2. PHP:FPM
 3. MariaDB:10.9.2
